# AI Domain Learning & Mentoring Chatbot (Free & Open-Source)

An AI-powered learning chatbot built using open-source Hugging Face models.
It helps users understand technical concepts with clear explanations,
examples, and practice questions across multiple domains.

This project is designed for interview preparation, self-learning,
and mentoring without using any paid APIs.

---

## Features
- Domain-aware mentoring (SQL, Python, QA, AI basics)
- Structured explanations with examples
- Practice questions for interview readiness
- Simple usage analytics (question frequency)
- 100% free and open-source

---
